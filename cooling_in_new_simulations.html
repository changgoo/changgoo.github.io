Temperature calculation¶

  1. Ideal gas law with $\gamma=5/3$. $$E \rightarrow e_{\rm int} \rightarrow P=(\gamma-1)e_{\rm int}$$
  2. Our density units are $[\rho]=\mu_H m_H{\rm cm}^{-3}$ to keep dimensionless density to the hydrogen number density. But, we may want to change $\mu_H=1.4$ (KKO11; KOK13; KO15b) to $1.4271$ for consistency.
  3. Simply adopt temperature dependent $\mu$ from SD93-CIE-$Z_\odot$. $$ T=\frac{\mu(T) m_H P}{\rho k_B} = \frac{\mu(T) P}{\mu_Hn_H k_B} $$
  4. Let's define $T_1\equiv m_H P/\rho k_B$.
  5. We now need to solve $$ f(T)\equiv T-\mu(T)T_1=0.$$ to get $T$ from any pair of $(\rho,P)$. Or, for a given $T_1$, we can find $T$ and $\mu(T)=T/T_1$.
  6. I'm going to first calculate $T$ vs. $T_1$ and tabulate it. I then tabulate cooling and heating as a function of $T_1$. In the code, $T$, $\Lambda$, and $\Gamma$ can be ontained by simple piecewise linear interpolation.

def calc_temp_i(T1,mumin=0.6182, mumax=1.295,toler=1.e-5):

    Tmax=mumax*T1
    Tmin=mumin*T1
    if T1 < 7.e3: return Tmax,0
    if T1 > 7.e6: return Tmin,0
    
    Tnew=T1
    for i in range(10):

        mui=muarr_SD[Tidx]
        muip1=muarr_SD[Tidx+1]
        Ti=tarr[Tidx]
        Tip1=tarr[Tidx+1]
        dmudti=(muip1-mui)/(Tip1-Ti)
    
        Tnew=T1*(mui-dmudti*Ti)/(1-dmudti*T1)
        
        dT=abs(Told/Tnew-1)
        if dT<toler: break
        
    return Tnew,i

Cooling Solver¶

$$ \frac{de}{dt}=-\rho\mathcal{L}(\rho,T)$$
  • Major collisional agents ($n_e$, $n_H$, $n_i$) may be different for low and high temperatures. But, we simply take net cooling rate as $n_H^2\Lambda(T)$ for all temperature.
  • For PE heating,
    • turn off at high temperature (ionized gas). (e.g., $2\times 10^4 {\rm K}$ or $\mu < 0.7$)
    • vertical dependence? (e.g., fixed scale height for integrated value?)
    • also use $n_H\Gamma$ for convenience.
  • It may be reasonable to assume that the temperature dependece of $\Gamma$ is similar to $\mu$: $$\Gamma=\Gamma_0\frac{\mu-\mu_{\rm min}}{\mu_{\rm max}-\mu_{\rm min}}+ \Gamma_{\rm CR,XR}$$
  • Here, $\Gamma_0=1.8\times10^{-26} {\rm erg/s}$ and $\Gamma_{\rm CR,XR}=2\times10^{-27} {\rm erg/s}$ (to set $\Gamma=2\times10^{-26}$ for atomic gas; Koyama & Inutsuka 2002).
  • We cut off the heating from $T=10^5K$.

Implicit solver with tabulated $\Lambda$ and $\Gamma$¶

  • It may be reasonable to assume that the temperature dependece of $\Gamma$ is similar to $\mu$: $$\Gamma=\Gamma_0\frac{\mu-\mu_{\rm min}}{\mu_{\rm max}-\mu_{\rm min}}+ \Gamma_{\rm CR,XR}$$
  • Using tabulated $\mu$, the same linear interpolation to $T^m$ for $T_i
  • Assume $\Lambda'(T^m)\approx \Lambda_i'$ and $\Gamma'(T^m)\approx \Gamma'_i$. $$T_1^{m+1}=\frac{T_1^n-\frac{\Delta t}{\tilde{k}}[n_H(\Lambda_i - \Lambda'_iT_i)-(\Gamma_i-\Gamma'_iT_i)]}{1+\frac{\Delta t}{\tilde{k}}\frac{T^m}{T_1^m}(n_H\Lambda'_i-\Gamma'_i)}$$

def get_Tidx(T,Tmin=10.,Tmax=1.e9,dT=0.01,nT=801):
    if type(T)==np.ndarray:
        Tidx=np.log10(T/Tmin)/dT
        Tidx[np.where(T<Tmin)]=0
        Tidx[np.where(T>=Tmax)]=nT-2
        return Tidx.astype(int)
    else:
        if T < Tmin: return 0
        if T >= Tmax: return nT-2
        Tidx=np.log10(T/Tmin)/dT
        return int(Tidx)

def get_temp(T1,t1arr,tarr):
    T1idx=get_Tidx(T1)
    Ti=tarr[T1idx]
    Tip1=tarr[T1idx+1]
    T1i=t1arr[T1idx]
    T1ip1=t1arr[T1idx+1]
    T=Ti+(Tip1-Ti)*(T1-T1i)/(T1ip1-T1i)
    
    return T
